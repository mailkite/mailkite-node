# MailKite examples — Node.js

Runnable, copy-pasteable examples. Each file's header comment lists what to set and `npm install`.

| File | What it shows |
| --- | --- |
| [`01-send-email.mjs`](01-send-email.mjs) | Send an email over a verified domain |
| [`02-receive-webhook.mjs`](02-receive-webhook.mjs) | Receive inbound mail as a webhook and **verify the HMAC signature** |
| [`03-agent-email-reply.mjs`](03-agent-email-reply.mjs) | **AI email agent** — inbound email → Claude drafts a reply → MailKite sends it, threaded |
| [`04-give-your-agent-an-inbox.mjs`](04-give-your-agent-an-inbox.mjs) | Give your agent its own address with MailKite's **built-in inbox agent** (no server) |
| [`05-server-login.mjs`](05-server-login.mjs) | **Server-side login + register** — your own account, or your users' accounts via OAuth |

Full docs: <https://mailkite.dev/docs> · AI agents: <https://mailkite.dev/docs/ai-agents>
